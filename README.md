# bt_api_bitunix

Bitunix exchange plugin for bt_api framework.

## Installation

```bash
pip install bt_api_bitunix
```

## Usage

```python
from bt_api_bitunix import BitunixExchangeDataSpot, BitunixRequestDataSpot
```

## Features

- REST API support
- Dual SHA256 signature authentication
- Symbol format: `BTC-USDT`
- Base URL: `https://fapi.bitunix.com`
